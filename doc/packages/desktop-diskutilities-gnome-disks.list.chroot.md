# GNOME disk utility

__manage and configure disk drives and media__

```

GNOME Disks is a tool to manage disk drives and media:

 * Format and partition drives.
 * Mount and unmount partitions.
 * Query S.M.A.R.T. attributes.

It utilizes udisks.

```
