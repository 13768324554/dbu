# GNOME network tools

__network information tool for GNOME__

```

GNOME Nettool is a network information tool which provides user interfaces for
some of the most common command line network tools including:
   * ifconfig
   * ping
   * netstat
   * traceroute
   * port scanning
   * DNS lookup
   * finger
   * whois

```
